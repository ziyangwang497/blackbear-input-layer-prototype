Adaptive Input Layer for Statement of Work Generation

A rule-based prototype that diagnoses assignment maturity, identifies missing or weak input, and prepares a structured package for SoW generation.

This prototype demonstrates the input-layer logic used in the thesis. It evaluates assignment information before SoW generation; it does not generate the final SoW.

                  5 Taxonomy-based scoring & evaluation metrics

                  This mode uses the taxonomy-based rubric for evaluation. Maturity screening is not used as the formal Chapter 6 evaluation metric.

                  The taxonomy scores are generated by the prototype’s rule-based diagnostic logic. Manual reference issues, if provided, are used only for evaluation comparison and do not affect the prototype’s own scoring.

                  Each of the 11 thesis taxonomy categories is scored automatically from the entered content: Captured = 1, Weak = 0.5, Missing = 0. Consistency warnings are counted separately and are not part of the completeness score.
